<h4>Technology used</h4>
<ul>

    <li>Redis (Cache)</li>
    <li>Flutter Secure Storage (Store User Token)</li>
    <li>Get It (Dependency Injection)</li>
    <li>Provider (State Management)</li>
    <li>Dio (HTTP Client)</li>
    <li>JWT (Authorization and Authentication)</li>
</ul>
